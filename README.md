# Healthcare Data Analysis & Predictive Modeling

This project performs an End-to-End Exploratory Data Analysis (EDA) and Predictive Modeling on a Healthcare Dataset using Python 

## Key Features & Analysis
** Disease-wise Analysis:** Grouped and analysed average billing amounts and hospital stay durations for various medical conditions.
Created a **Bar Graph** to analyse average costs per medical condition.
Plotted a **Pie Chart** for patient gender distribution (perfectly balanced at 50% Male / 50& Female).
Generated a ** Histogram with KDE** to view the patient age distribution .
**Machine Learning:**Built a **Random Forest Classifier** model using 'scikit-learn' to predict medical conditions based on patient demographics(Age, Gender, Blood Type).

## Tech Stack
**Language:** Python
**Libraries:** Pandas, NumPy, Matplotlib, Seaborn, Scikit-Learn
**Platform:** Kaggle Notebook
